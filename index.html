Installation

Cài đặt lần đầu — plugin tự tạo mọi thứ

Plugin tự động khởi tạo toàn bộ cấu trúc thư mục khi lần đầu khởi động:

# 1. Build JAR mvn clean package # 2. Copy vào server cp target/LivingWorldSMP-7.0.0.jar plugins/ # 3. Khởi động server — plugin tự tạo: # plugins/LivingWorldSMP/ # ├── config.yml # ├── database/world.db # ├── backups/ # └── logs/
✅
Zero config: Mặc định hoạt động ngay. Chỉnh config.yml sau khi đã chạy thử.
⚠️
Yêu cầu: Paper 1.21.x (không hỗ trợ Spigot thuần), Java 21+, ít nhất 2GB RAM.

Build requirements:

Maven 3.9+ Java 21 JDK Paper API 1.21.4 SQLite JDBC (shaded)

Bug Fix — DataManager.java

Root cause và cách khắc phục
🐛
Lỗi biên dịch V6:
DataManager.java:[1,1] unnamed class should not have package declaration

Root cause: Java 21 có tính năng "unnamed class" (preview) — cho phép viết code mà không có khai báo class. Nhưng nếu file có package declaration mà không có class DataManager { } wrapper, Java 21 hiểu đây là unnamed class và từ chối package declaration.

// ❌ V6 — DataManager.java bị lỗi: package dev.livingworld.data; // ← dòng 1 // ... imports ... // THIẾU: public class DataManager { } ← Java 21 báo lỗi tại [1,1] Connection conn; // code không có class wrapper → unnamed class
// ✅ V7 — DataManager.java đúng: package dev.livingworld.data; // package declaration import java.sql.*; // ... other imports ... public final class DataManager { // ← class declaration BẮT BUỘC private Connection connection; // ... implementation ... }

Ngoài ra, pom.xml V7 thêm -proc:none để tắt annotation processing warning:

<compilerArgs> <arg>-proc:none</arg> <!-- suppress AP discovery warning --> </compilerArgs>

Performance System

V7 — Ưu tiên #1
🏗️ WorkerPool

Thread pool cố định N threads (config: worker-threads). Tất cả DB query, simulation, backup chạy async tại đây. Tên thread: lw-worker-1, lw-worker-2…

V7 New
📋 JobQueue

Priority queue (HIGH / NORMAL / LOW). Khi queue đầy, các LOW-priority job bị loại bỏ trước. HIGH-priority job (DB save, economy) không bao giờ bị drop.

V7 New
⏱️ PriorityScheduler

Chạy trên main thread Bukkit, dispatch tối đa N task/tick. Dùng cho thao tác cần main thread (spawn entity, block change).

V7 New
💨 CacheLayer

TTL + LRU cache generic. Tự evict expired entries. Dùng cho balance cache, nation lookup. Hit rate hiển thị trong /lw profiler.

V7 New
🗺️ ChunkSimulator

Chỉ mô phỏng chunk đang loaded và nằm trong radius N chunks quanh player. Round-robin để tránh starvation. Max chunk/tick có thể giới hạn.

V7 New
💿 AsyncDatabase

SQLite với WAL mode + PRAGMA synchronous=NORMAL. Tất cả query wrap qua CompletableFuture. Không có blocking call trên main thread.

V7 New
💡
Mục tiêu TPS ổn định: ≥18.0 với 100–200 player. Dưới ngưỡng tps-warn-threshold, /lw profiler hiện màu đỏ.

Logging System

Named loggers — bật/tắt từng loại

V7 thay thế getLogger().info(...) bằng named loggers chuyên biệt:

📊 EconomyLogger

Log transaction, price change. Tắt: /lw log economy off

🌍 SimulationLogger

Log chunk tick, world events. Tắt: /lw log simulation off

Error Recovery

Module isolation — một lỗi không tắt cả plugin

ModuleManager wrap mỗi module trong try-catch độc lập:

// Ví dụ: Economy lỗi khi enable try { economyModule.onEnable(); // → throws RuntimeException status = ENABLED; } catch (Throwable t) { status = ERROR; // Economy → ERROR errors.put("economy", t); // lưu để /lw status hiển thị // Nation, Boss, Quest, History vẫn tiếp tục enable }

Kết quả trên console khi Economy lỗi:

[LW/CORE] ✓ Module enabled: History [LW/CORE] ✗ Module economy failed to enable: Database error [LW/CORE] ✓ Module enabled: Nation [LW/CORE] ✓ Module enabled: Boss [LW/CORE] Modules: 5/6 enabled, 1 failed. [LW/CORE] 1 module(s) failed to start — see above. Others are unaffected.
💡
Dùng /lw status để xem trạng thái từng module. Module ERROR hiện màu đỏ kèm error message.

backup: enabled: true interval-hours: 24 # mỗi 24h backup tự động keep-days: 7 # giữ 7 file gần nhất directory: backups

Economy Module

Số dư, chuyển tiền, thị trường giá động
💳 Balance

Mỗi player có tài khoản. Số dư ban đầu: 500⬡. Cache TTL 5 phút, persist async về SQLite.

🔄 Transfer

Chuyển tiền với thuế (mặc định 5%). Giao dịch được log đầy đủ với timestamp.

📈 Dynamic Market

Giá thị trường thay đổi theo supply/demand. Nhiều buy → giá tăng. Nhiều sell → giá giảm. Cập nhật mỗi 5 phút.

🏦 Nation Treasury

Quốc gia có kho bạc riêng. Thuế từ giao dịch có thể được config để vào kho quốc gia.

👑

Nation Module

Tạo vương quốc, ngoại giao, chiến tranh
🏰 Nation Creation

Player tạo quốc gia với tên độc nhất. Người tạo là Leader. Load toàn bộ nation vào memory khi startup.

🤝 Diplomacy

3 trạng thái: ALLY / NEUTRAL / ENEMY / WAR. Được persist vào DB, hiển thị trong nation info.

💬 Nation Chat

Tên quốc gia tự động hiển thị trước chat message của member. [Kingdom East] Steve: ...

⚡ Power System

Mỗi quốc gia có Power Score (tăng theo member, chiến thắng). Hiển thị trong /lwnation list.

🐉

Boss Module

Boss có trí nhớ, học hỏi, thích nghi

Boss nhớ ai tấn công, vũ khí nào được dùng nhiều nhất, và thay đổi chiến thuật:

🧠 Boss Memory

Mỗi lần boss chết, dữ liệu được persist: damage/player, weapon usage. Boss lần sau spawn biết đây là player nguy hiểm.

⚡ Phase System

HP threshold triggers phase change. Mỗi phase mở khóa abilities mới. Dragon: Phase 1→2→3 (75%→50%→25%).

🎯 Ability Cooldowns

Mỗi ability có cooldown độc lập. Dragon fireball: 5s CD. Shadow Lich wither: 8s CD.

📊 Boss Bar

BossBar hiển thị HP%. Màu thay đổi: xanh (>50%) → vàng (>25%) → đỏ (<25%).

Boss Types:

ANCIENT_DRAGON — 500 HP — Phases: 75/50/25% IRON_TITAN — 300 HP — Phases: 66/33% SHADOW_LICH — 250 HP — Phases: 50% VOLCANIC_GOLEM — 400 HP — Phases: 60/30% STORM_SERPENT — 350 HP — Phases: 70/40/10%

Quest Module

Quest động, sinh ra từ trạng thái thế giới

Quest không cố định. AI phân tích world state và sinh quest phù hợp:

🏘️ Nation Resource Quest

Nation thiếu gỗ → sinh quest "Kingdom East cần 64 OAK_LOG". Player hoàn thành → Nation phát triển.

📈 Market Quest

Giá Iron cao → sinh quest "Gather 32 Iron Ingot". Reward scale theo market price.

⚔️ Boss Kill Quest

Boss đang sống → sinh quest "Defeat Ancient Boss". Reward 500⬡.

⏰ Expiry

Mỗi quest có TTL. Quest GATHER: 24h. Quest BOSS: 6h. Hết hạn tự biến mất khỏi pool.

History Module

World Chronicle — thế giới kể câu chuyện

Mọi sự kiện lớn được ghi vào lịch sử thế giới. Player nhận sách Chronicle:

⚔ Year 12: Ancient Dragon appeared in the world! 👑 Year 12: Steve founded Kingdom of East. 💰 Year 13: Iron prices rising. 🌾 Year 14: Crop failure reported in the plains! ⚔ Year 15: Ancient Dragon was slain by Alex! 📜 Year 15: Alex completed: Defeat Ancient Boss.

Lệnh lấy sách Chronicle: /lwhistory chronicle

World year tự tăng mỗi 24000 ticks (1 Minecraft day = 1 world year).

💡
Mọi module đều gọi historyModule.recordEvent(...) — nên Chronicle phản ánh toàn bộ thế giới tự động.

Public API

Tích hợp từ plugin khác

Plugin khác có thể truy cập LivingWorldSMP qua LivingWorldAPI:

// Trong plugin của bạn: Plugin lw = Bukkit.getPluginManager().getPlugin("LivingWorldSMP"); if (lw instanceof LivingWorldPlugin lwp) { LivingWorldAPI api = lwp.getAPI(); // Economy api.getBalance(uuid).thenAccept(bal -> ...); api.addBalance(uuid, name, 100.0); double price = api.getMarketPrice("DIAMOND"); // Nation Optional<Nation> nation = api.getNationByPlayer(uuid); Collection<Nation> all = api.getAllNations(); // History — plugin của bạn có thể ghi sự kiện vào Chronicle api.recordHistoryEvent("CUSTOM", "Title", "Description"); // Thông tin boolean economyOn = api.isModuleEnabled("economy"); int year = api.getWorldYear(); }
